Welcome to GBC Tech Challenge.

This is a simple 2-part exercise that will help us evaluate your capabilities.

The first part is a HTML/CSS page that is intentionally broken. The goal is to identify and fix the code. This is a real-world situation that you are likely to face in the tech industry. Be mindful that there are definite code errors and mixed syntax.

The second part of the exercise involves creating a matching "Contact" page with a form that users can use to send a quick message.
